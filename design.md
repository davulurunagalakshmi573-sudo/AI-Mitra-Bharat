## Overview
AI Mitra Bharat is an AI-powered multilingual assistant designed to bridge the digital divide in India by providing easy access to government services, healthcare guidance, and public information through voice-first interactions.

## Architecture
The system follows a hybrid cloud + edge architecture:

1. User interacts via WhatsApp or Android app
2. Voice/text processed using NLP models
3. AI engine fetches data from:
   - Government APIs
   - Healthcare datasets
   - Local knowledge base
4. Response delivered via voice or text

## Core Components
- Multilingual NLP Engine
- Speech-to-Text and Text-to-Speech
- Conversational AI chatbot
- Knowledge aggregation layer
- Cloud backend services

## Platforms
